# Bio and Neural Rights Website

This repository contains a GitHub Pages website dedicated to a three-part series on Bio and Neural Rights. The website includes articles, an about page, and a flyer.

## Project Structure

- **index.html**: The main entry point of the website.
- **about.html**: Information about the website and the author.
- **articles/**: Contains the three articles in the series.
  - **article-1.html**: The first article on Bio and Neural Rights.
  - **article-2.html**: The second article on Bio and Neural Rights.
  - **article-3.html**: The third article on Bio and Neural Rights.
- **assets/**: Contains all the assets for the website.
  - **css/**: Contains the CSS styles.
    - **style.css**: Styles for the website.
  - **js/**: Contains JavaScript files.
    - **script.js**: JavaScript for interactivity.
  - **flyer/**: Contains the flyer.
    - **flyer.pdf**: The flyer providing additional information.
- **_config.yml**: Configuration file for the website.
- **README.md**: Documentation for the project.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the repository directory.
3. Open `index.html` in a web browser to view the homepage.
4. Access the articles through the links provided on the homepage.

## Custom Domain Setup

Once the website is ready, you can set up GitHub Pages with a custom domain by following these steps:

1. Go to the repository settings on GitHub.
2. Scroll down to the "GitHub Pages" section.
3. Enter your custom domain in the "Custom domain" field.
4. Configure your DNS settings to point to GitHub's servers.
5. Save the changes and wait for the DNS to propagate.

For more detailed instructions, refer to the [GitHub Pages documentation](https://docs.github.com/en/pages).
